1. 返回解引用的this.


    在一些时候，我们定义的类操作符将会对对象进行类似运算符的操作，这需要成员函数返回一个左值。这样的函数一般采用如下形式定义： 

       class T{
   public:
   T& f(){return *this;};
   };

    值得注意的是，const对象是无法作为左值的，这也就意味着const对象无法调用这类函数;
    通常，对于const对象需要进行重载操作，对于如上的函数f，应重载为以下形式：
    T::T T::f()const{return *this};

  2. 构造函数初始化问题


    未赋初值的变量将初始化为默认值，但某些变量必须赋初值：const对象 引用对象 没有定义默认构造函数的类对象
    以上对象的初始化也并不能在构造函数体内部赋值，而应采用c++11标准制定的构造函数初始化方法：

           class T{
       public:
       T(int i):a(i),b(i){};
       private:
       const int a;int& b;};
